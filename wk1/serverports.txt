Q1:What is the definition of a server port?
A: Server ports are the logical endpoints of a network connection that is used to exchange information between a server and a web client.

Q2: What is server port range?
A: Port numbers range from 0 to 65536.

Q3: How many bits does number above represent?
A: 16 Bit Integer
